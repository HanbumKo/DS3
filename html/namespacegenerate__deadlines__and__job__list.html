Detailed Description

This file generates a csv file with the deadlines for each snippet, following the specified thresholds, and it generates a job_list with a list of snippets (defined by num_snippets).

More information on the deadline generation can be found in HiLITE (https://ieeexplore.ieee.org/document/9085952).
