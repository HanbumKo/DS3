Detailed Description

The MergeMethod class is an Enum that can be used to select which merge method to use to merge DAGs together when passed in to merge_dags.

There are two different merge methods available.

  • COMMON_ENTRY_EXIT: merge graphs by directly connecting a dummy head and sink node between all heads and sinks of the incoming DAGs
  • RANKING_BASED: utilize ranking-based DAG merging where DAGs are merged by linking their head nodes with nodes in the other graph that have comparable rank values
    See also: heft.RankMetric
